# pyFishTank
Application to store update fish and fish tank information such as food, feeding, etc.

# Usage
Application has Console Branch with json file support and Console-SQLite Branch with SQLLite useage. 

Uses native python libraries. Build Targets 3.12.X

Added React for UI. 

# To Run Locally
Start the API server:


source .venv/bin/activate
python main.py
In a separate terminal, start the frontend:


cd frontend
npm run dev
Open the frontend URL shown in the terminal (typically http://localhost:5173)
